ruuvi_driver_status_t task_i2c_init ( void  )

Initialize I2C bus.

This function looks up the I2C definitions from ruuvi_boards.h and initializes the bus and relevant GPIO pins.

Returns
RUUVI_DRIVER_SUCCESS on success
RUUVI_DRIVER_ERROR_INTERNAL if I2C pins cannot be pulled high
RUUVI_DRIVER_ERROR_INVALID_STATE if I2C is already initialized
Warning
asserts (reboot / break on debugger) if SCL pin equals SDA pin
